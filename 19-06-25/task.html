Difference between == and ===

'==' is Loose Equality used to check Value. '===' is Strict Equality used to check Value + Type

== Result (True)

'5' == 5 ‚Üí true

== Result (False)

'5' == 6 ‚Üí false

=== Result (True)

5 === 5 ‚Üí true

=== Result (False)

'5' === 5 ‚Üí false
